Detailed Description

Class with implementation of adjacency matrix structure.

Implemented, as a square matrix of logical value, in which element with i_th row and j_th column equals true, when vertices i and j are connected. Otherwise, false.

The order of the adjacency matrix is equal to maximum count of vertices in graph
